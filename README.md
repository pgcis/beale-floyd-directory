# Floyd KC Program Directory

Interactive program directory widget for the Beale Infrastructure Floyd KC engagement.

Hosted on GitHub Pages and embedded via iframe into Google Sites.

## Data Source

This widget reads from a published Google Sheet. Edit the Sheet to update content - no code changes needed.

## Deployment

Pushes to `main` auto-deploy to GitHub Pages at `pgcis.github.io/beale-floyd-directory/`.

## Confidential

This widget renders team contact information. Access is controlled via Google Sites sharing settings, not at this layer.
